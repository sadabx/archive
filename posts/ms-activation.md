## 📖 About

This repository serves as a documentation and usage guide for the **Microsoft Activation Scripts (MAS)**. It demonstrates how to use PowerShell to manage activation states for Windows and Office products using the open-source method provided by `massgrave.dev`.

> **⚠️ Disclaimer:**
> This guide is for **educational and research purposes only**. It allows system administrators to test activation scenarios. The author of this repository is not affiliated with Microsoft or the MAS project developers. Please support software developers by purchasing genuine licenses.

## 🚀 Quick Start (The "One-Line" Method)

You do not need to download files manually. The entire toolkit is accessible via a single PowerShell command that fetches the latest script from the official repository.

### 1. Open PowerShell

Search for **PowerShell** in your Start Menu, right-click it, and select **Run as Administrator**.

### 2. Execute the Command

Copy and paste the following command into the terminal and press **Enter**:

```powershell
irm [https://massgrave.dev/get](https://massgrave.dev/get) | iex
```
